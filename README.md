# Just Create Flashcard Memo for Preparing Japanese Quiz :D

A simple flashcard memo application to help you prepare for Japanese quizzes. You can review them to enhance your learning.

## Installation

1. Clone this repository:
```bash
git clone https://github.com/jjuaon/NihonGO.git
```
   
2. Navigate into the project directory:
```bash
cd .\NihonGO\
```

3.Install and run:
```bash
npm install
npm run dev
```
